CSS Grid vs Flexbox: Khi Nào Dùng Cái Nào?

10 Tháng 11, 2024 CSS 7 phút đọc

Giới Thiệu

CSS Grid và Flexbox đều là những công cụ layout mạnh mẽ được giới thiệu vào CSS3. Nhưng khi nào bạn nên dùng cái nào? Bài viết này sẽ giúp bạn hiểu rõ sự khác biệt và cách chọn công cụ phù hợp cho từng trường hợp.

Flexbox là gì?

Flexbox (Flexible Box Layout) là một 1-dimensional layout model. Nó tốt cho việc sắp xếp items theo một hàng hoặc một cột: 

.container {
  display: flex;
  gap: 1rem;
}

.item {
  flex: 1;
}

CSS Grid là gì?

CSS Grid là một 2-dimensional layout model. Nó cho phép bạn kiểm soát cả hàng và cột cùng một lúc: 

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1rem;
}

Khi Nào Dùng Flexbox?

  • Sắp xếp items theo một chiều (hàng hoặc cột)
  • Navigation menu
  • Centered layout
  • Space between items

Khi Nào Dùng CSS Grid?

  • Layout phức tạp với nhiều hàng và cột
  • Dashboard layouts
  • Magazine-style layouts
  • Responsive grid layouts

Ví Dụ Thực Tế

Sử dụng Flexbox cho navigation:

nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Sử dụng Grid cho dashboard:

.dashboard {
  display: grid;
  grid-template-columns: 200px 1fr 300px;
  grid-template-rows: 60px 1fr;
  gap: 1rem;
}

Kết Luận

Cả Flexbox và Grid đều hữu ích. Hãy chọn dựa trên nhu cầu layout của bạn. Thường thì bạn sẽ sử dụng cả hai trong cùng một dự án!
