Web Performance: Tối Ưu Hóa Tốc Độ Trang Web

5 Tháng 11, 2024 Backend 8 phút đọc

Tại Sao Performance Quan Trọng?

Hiệu năng web ảnh hưởng trực tiếp đến trải nghiệm người dùng và SEO. Một trang web chậm có thể khiến người dùng rời đi, dẫn đến tăng bounce rate và giảm conversion.

Lazy Loading Images

Một trong những cách tốt nhất để cải thiện performance là lazy loading hình ảnh: 

<img src="image.jpg" loading="lazy" alt="Description" />

Code Splitting

Chia nhỏ code của bạn để chỉ tải những gì cần thiết:

import React, { Suspense, lazy } from 'react';

const HeavyComponent = lazy(() => import('./HeavyComponent'));

export default function App() {
  return (
    <Suspense fallback={<div>Loading...</div>}>
      <HeavyComponent />
    </Suspense>
  );
}

Caching

Sử dụng browser caching và service workers để giảm request:

// Cache API response
const cacheResponse = async (request) => {
  const cache = await caches.open('my-cache');
  const response = await cache.match(request);
  
  if (response) return response;
  
  const fetchResponse = await fetch(request);
  cache.put(request, fetchResponse.clone());
  return fetchResponse;
};

Minify CSS và JavaScript

Sử dụng build tools như webpack hoặc rollup để minify code:

  • Giảm kích thước file
  • Tăng tốc độ download
  • Cải thiện parse time

Core Web Vitals

Chú ý đến 3 metrics chính từ Google:

  • LCP (Largest Contentful Paint): < 2.5s
  • FID (First Input Delay): < 100ms
  • CLS (Cumulative Layout Shift): < 0.1

Tools để Đo Lường

  • Google PageSpeed Insights
  • Lighthouse
  • WebPageTest
  • GTmetrix

Kết Luận

Tối ưu hóa performance là quá trình liên tục. Hãy đo lường, phân tích, và cải thiện thường xuyên. Người dùng sẽ cảm ơn bạn!
