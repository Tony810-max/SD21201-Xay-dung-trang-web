Hướng Dẫn React Hooks Cho Người Mới Bắt Đầu

15 Tháng 11, 2024 React 5 phút đọc

Giới Thiệu

React Hooks là một tính năng mạnh mẽ được giới thiệu trong React 16.8 cho phép bạn sử dụng state và các tính năng khác của React mà không cần viết class component. Hooks đã thay đổi cách chúng ta viết React code, làm cho nó trở nên đơn giản hơn và dễ bảo trì hơn.

useState Hook

useState là hook cơ bản nhất. Nó cho phép bạn thêm state vào functional component: 

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
    </div>
  );
}

Trong ví dụ này, chúng ta sử dụng useState để tạo một state variable count. setCount là function để update state này.

useEffect Hook

useEffect cho phép bạn thực hiện side effects trong functional components. Nó hoạt động như componentDidMount, componentDidUpdate, và componentWillUnmount được kết hợp: 

import React, { useEffect, useState } from 'react';

function User() {
  const [user, setUser] = useState(null);

  useEffect(() => {
    // Fetch user data
    fetch('/api/user')
      .then(res => res.json())
      .then(data => setUser(data));
  }, []); // Empty dependency array means run once

  if (!user) return <div>Loading...</div>;
  
  return <div>{user.name}</div>;
}

Custom Hooks

Bạn cũng có thể tạo custom hooks để reuse logic giữa các components: 

function useFetch(url) {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    fetch(url)
      .then(res => res.json())
      .then(data => {
        setData(data);
        setLoading(false);
      })
      .catch(err => {
        setError(err);
        setLoading(false);
      });
  }, [url]);

  return { data, loading, error };
}

Kết Luận

React Hooks là một cách tuyệt vời để viết React code sạch và dễ bảo trì. Bắt đầu với useState và useEffect, sau đó khám phá các hooks khác như useContext, useReducer, v.v. Hãy thực hành và tạo các custom hooks để reuse logic trong ứng dụng của bạn.
